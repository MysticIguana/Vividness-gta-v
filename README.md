### Detailed Installation Steps

**Step 1: Prepare Your Game**
- Update GTA V to the latest version through Steam or Rockstar Launcher
- Create a backup of your current `mods` folder (if you have existing mods)
- Make sure the game runs normally before installing

**Step 2: Install OpenIV**
- Download and install OpenIV from the official website
- Open OpenIV and select Grand Theft Auto V as your game
- Click "Tools" → "ASI Manager" → Install "ASI Loader"

**Step 3: Install Script Hook V (Optional but Recommended)**
- Download Script Hook V from the official website
- Copy `ScriptHookV.dll` to your GTA V directory
- Some Vividness features require Script Hook V

**Step 4: Download and Install Vividness**
- Click the download button above
- Extract the downloaded archive using WinRAR or 7-Zip
- Open OpenIV and enable "Edit Mode"
- Navigate to your GTA V directory in OpenIV
- Drag and drop the Vividness files into the appropriate locations

**Step 5: Configure In-Game Settings**
- Launch GTA V
- Go to Settings → Graphics
- Set DirectX Version to **DX11**
- Set Shader Quality to **High** or **Very High**
- Set Shadow Quality to **High** or **Very High**
- Set Reflection Quality to **High** or **Ultra**
- Set Grass Quality to **High** or **Ultra**
- Set Post FX to **High** or **Ultra**
- Turn **on** High Detail Streaming When Flying

**Step 6: Adjust Brightness**
- Go to Settings → Display
- Set Brightness to your preference (50% recommended)
- Adjust contrast if needed
